Contaq jQuery plugin

Contaq is simple jQuery plugin, to get feedback from users. To see it in action simply click tab on the left.

Usage

Download zip file with plugin from here or visit github page of the project. After download add code to your website: 

<script type="text/javascript" src="http://code.jquery.com/jquery-1.7.1.min.js"></script>
<script type="text/javascript" src="/contaq.js"></script>
<link rel="stylesheet" type="text/css" href="/contaq.css"/>

<script>
  $(document).ready( function() {
    $.contaq();
  });
</script>

It is also possible to pass options to plugin, default options are: 

{
  subjectText: "Subject",
  emailText: "Email",
  bodyText: "Your message",
  submitText: "Send",
  postUrl: "/post",
  successText: "Success!",
  failureText: "Failure",
  tryAgainText: "Try again",
  waitText: "Please wait..."
}

Main option you will be interested in is postUrl, which by default is /post.
